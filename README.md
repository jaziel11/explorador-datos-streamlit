## Descripción
Esta aplicación permite cargar datasets en formatos CSV, Excel y JSON para realizar análisis exploratorio y visualización de datos de manera interactiva.

## Funcionalidades
- Carga de archivos (CSV, Excel, JSON)
- Vista previa del dataset
- Estadísticas descriptivas
- Análisis de valores faltantes
- Matriz de correlación
- Filtros dinámicos por columna
- Visualizaciones interactivas:
  - Líneas
  - Barras
  - Dispersión
  - Histograma
  - Diagrama de caja
- Exportación del dataset filtrado

## Tecnologías utilizadas
- Python
- Streamlit
- Pandas
- Plotly
